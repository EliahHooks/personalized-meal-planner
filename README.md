# Personalized Meal Planner

## Team

- Eliah Hooks
- Andrew Brickweg

A web app for browsing and planning personalized meals, built with PHP and MySQL.

## Features

- User registration and login
- Browse and search meals
- Build and manage a personal meal plan
- Admin panel to manage meals and users

## Requirements

- XAMPP (or any Apache + PHP + MySQL stack)
- PHP 7.4+
- MySQL 5.7+

## Setup

1. Clone the repo and move it to your XAMPP `htdocs` folder
2. Start Apache and MySQL in the XAMPP Control Panel
3. Create a new database in phpMyAdmin and import the `.sql` file from the `sql/` folder
4. Update the database credentials in the `database/` folder
5. Visit `http://localhost/personalized-meal-planner/` in your browser

## Admin Panel

Access the admin panel at `/admin/` to manage meals and users. Requires an admin account to be set up in the database first.
